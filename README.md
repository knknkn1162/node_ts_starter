# environment

```
yarn add global typescript
yarn add tslint
yarn add webpack webpack-cli ts-loader -D
# transform ./src/**/*.ts to ./dist/**/*.js
# you can append shortcut command in scripts
yarn init # create package.json
tsc --init # create tsconfig.json
tslint --init # create tslint.json
```

yarn run test
